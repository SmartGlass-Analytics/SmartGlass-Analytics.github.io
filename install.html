How to Install SmartGlass

SmartGlass is a mobile application that allows real-time basketball stats to be displayed on Brilliant Labs Frame smart glasses. It integrates Synergy API data with a Bluetooth-connected Android tablet.

Requirements and Sources

  • Android Studio (Electric Eel or newer) - https://developer.android.com/studio
  • Python 3.10+ - https://www.python.org/downloads/release/python-3100/
  • Elo M51 Android Tablet - Project Sponsor
  • Brilliant Labs Frame glasses - Project Sponsor - https://docs.brilliant.xyz/frame/frame-sdk/
  • Synergy API credentials - Project Sponsor

Steps

1. Clone the Repositories

git clone https://github.com/SmartGlass-Analytics/smartglass
cd smartglass

2. Set up the Python Interpreter

  • Ensure that proper Python version is installed on your device per https://chaquo.com/chaquopy/doc/current/versions.html
  • Follow the first repsonse on https://stackoverflow.com/questions/54667573/how-to-configure-python-interpretor-in-android-studio to configure python interpreter before building

    • 3. Build the Android App

    1. Open smartglass/app in Android Studio
    2. Connect Elo M51 tablet
    3. Go To Build/Make Project
    4. Allow Bluetooth & Location permissions

    4. Pair with Frame Glasses

  • Make sure glasses say they are ready to pair and connect with phone's bluetooth to pair with device
  • You are now ready to begin testing!
