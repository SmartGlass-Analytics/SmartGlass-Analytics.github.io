How to Install SmartGlass

SmartGlass is a mobile application that allows real-time basketball stats to be displayed on Brilliant Labs Frame smart glasses. It integrates Synergy API data with a Bluetooth-connected Android tablet.

Requirements

  • Android Studio (Electric Eel or newer)
  • Python 3.10+
  • Elo M51 Android Tablet
  • Brilliant Labs Frame glasses
  • Bluetooth 5.3 support
  • Synergy API credentials

Steps

1. Clone the Repositories

git clone https://github.com/SmartGlass-Analytics/smartglass
cd smartglass

2. Build the Android App

  1. Open smartglass/app in Android Studio
  2. Connect Elo M51 tablet
  3. Click Run ▶️
  4. Allow Bluetooth & Location permissions

3. Set Up Python Script

pip install -r requirements.txt
python apipractice.py
