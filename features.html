Features Guide for SmartGlass Analytics

Welcome to the SmartGlass Analytics documentation! This guide explains all available features in the SmartGlass App and how they work.

Project Overview

SmartGlass Analytics is a mobile application that allows real-time basketball statistics to be displayed on Brilliant Labs Frame smart glasses. It integrates live data pulled from the Synergy API into an Android tablet application, which sends information to the glasses over a Bluetooth Low Energy (BLE) connection.

The application uses a hybrid architecture:

  • Python (via Chaquopy) inside the Android app fetches and processes basketball statistics from Synergy.
  • Kotlin handles the Android user interface (UI) and Bluetooth communication.
  • Bluetooth LE is used to transmit selected stats as formatted text to the smart glasses.
  • Lua scripting on the Brilliant Labs Frame renders the text display on the glasses.

Core Features

1. Real-Time Basketball Stats Fetching

The app pulls live player statistics from the Synergy API using Python code embedded in the app (via Chaquopy). Data includes points, assists, rebounds, field goal percentage, and more, parsed and displayed in the app.

2. Dynamic Player Profiles

The app provides a player directory for easy navigation. Users can select any Alabama player to view detailed game statistics.

3. Select and Send Stats to Smart Glasses

After selecting a player, users can choose specific statistics (such as points, rebounds, assists) by checking boxes on the player's profile screen. When the "Send to Glasses" button is clicked:

  • Kotlin reads the selected stats and triggers a Python function inside the app.
  • Python formats the message into a compact, readable string.
  • The message is sent over Bluetooth LE to the Brilliant Labs Frame glasses.
  • Lua scripting on the glasses renders the received text onto the heads-up display.

No manual typing is required by the user once selections are made. The whole pipeline is automated.

4. Bluetooth Low Energy (BLE) Connection

The app automatically handles Bluetooth pairing and communication with the Brilliant Labs Frame. Glasses must first be put into "Ready to Pair" mode, and permissions (Bluetooth and Location) must be granted on the tablet.

5. Lightweight and Expandable UI

The app uses Jetpack Compose for a modern, responsive UI:

  • Home Screen: Navigation hub to player stats.
  • Player Stats Screen: Browse player profiles and game performance.
  • Player Profile Screen: Select specific stats and send to glasses.
Future features like lineup-based stats, shot charts, or advanced analytics can be added easily.

6. Offline Testing and Debugging

Unit testing scripts exist for verifying API calls and Bluetooth message formatting in Python without needing the glasses connected.

Technologies and Architecture

  • Kotlin — Main Android app development (UI and Bluetooth management).
  • Python 3.10+ — Embedded via Chaquopy for API communication and message formatting.
  • Gradle — Build system for Android Studio project.
  • Brilliant Labs Frame SDK — Glasses display connection and Lua scripting interface.
  • Synergy Sports API — Live basketball statistics provider.

Planned Improvements

  • Improve Bluetooth reconnection stability.
  • Enhance text formatting for even better readability on glasses.
  • Add color-coded stats (e.g., green for strong performance) on glasses display.
  • Integrate lineup-based statistical summaries in the UI.
