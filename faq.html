Frequently Asked Questions (FAQs)

❓ How do I connect the Frame glasses?

Turn on the glasses, ensure Bluetooth is enabled on the tablet, and tap the "Connect" button within the app. The device will automatically scan and pair with available Frame devices.

❓ What if the stats do not show up on the glasses?

  • Ensure apipractice.py is running and outputting a fresh data file
  • Make sure the tablet is reading from the correct location (e.g., internal storage)
  • Verify that the BLE connection was successful (watch for the "connected" message in logs)

❓ What permissions are required on the tablet?

The Android app requires the following runtime permissions:

  • BLUETOOTH
  • BLUETOOTH_ADMIN
  • BLUETOOTH_CONNECT (Android 12+)
  • ACCESS_FINE_LOCATION

❓ Can I run the app on a different tablet?

Yes. Make sure the tablet supports Bluetooth 5.0+ and runs Android 10 or later. Enable all necessary permissions, and deploy the app via Android Studio or install a signed APK.

⚠️ Gotchas & Known Issues

  • Some BLE UUIDs for the Frame device may change — use nRF Connect to scan if needed
  • Synergy API requires valid authentication headers in SynergyAuth.txt
  • The Frame display has limited screen size, so concise stats formatting is important
