Project Goals

Our project aims to develop a smart glass analytics system that displays real-time data and analytics on a smart glasses user interface related to viewing of game film, practice film, or live game in order to support quicker in-game decision making.

    Welcome to SmartGlass

    SmartGlass is a capstone project that delivers real-time basketball statistics to Brilliant Labs Frame smart glasses using an Android tablet and Synergy Sports data. This documentation will guide users and developers through installation, usage, extension, and troubleshooting of the project.
